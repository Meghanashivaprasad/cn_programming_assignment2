1. Open Terminal on Linux
2. Execute the following commands to compile server.c and client.c
	
	$> gcc -o client client.c
	$> gcc -o server server.c

3. Execute the following commands to run the exe server and client files after compiling on two different terminals
	
	# on terminal 1
	$> ./server


	# on terminal 2
	$> ./client

4. Please refer the output images directory for output images screenshots
